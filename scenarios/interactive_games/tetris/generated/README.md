# Tetris (Spec Driven Generated)

## 主な機能

- 7 種のテトリミノ (I,O,T,S,Z,J,L) / 7 バッグランダム
- 回転 (簡易 SRS キック)
- ホールド (1 ピース毎に 1 回)
- ネクスト 3 個表示
- ゴーストピース表示
- ソフト / ハードドロップ
- レベル & スコア & 消去ライン表示 (10 ライン毎にレベルアップ / 落下速度上昇)
- ハイスコア (ローカルストレージ保存 上位 10 件)
- タイトル / ハイスコア / 遊び方 シーン遷移
- レスポンシブ対応 (幅が狭い場合は縦配置)
- 矢印キー / スペース / C(Shift) / P(Esc) 入力、スクロール抑止
